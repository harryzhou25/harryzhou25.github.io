背包问题汇总

2023-09-10 动态规划

背包问题

背包问题原型是从nnn个带有体积物体的物体中选取一部分放入背包，使背包中的物体达到最大的价值。
也可以转化为 在nnn个物体中选取一部分进行组合，使价值达到一个指定的值。
因此背包问题也可以理解为组合问题

01背包

从给定的nnn个体积为www，价值为vvv的物品中，选取一部分装入带有容量限制的背包，使得背包中的物品价值最大。
思路：定义dp[i][j]dp[i][j]dp[i][j]表示在前iii个物品中，容量为jjj的背包最大能达到的价值。
转移方程为

dp[i][j]=max(dp[i−1][j],dp[i−1][j−w]+v) dp[i][j] = max(dp[i-1][j], dp[i-1][j-w]+v) dp[i][j]=max(dp[i−1][j],dp[i−1][j−w]+v)

其中www和vvv分别表示当前第iii个的物品的重量和价值。
模板：

    for(int i = 0; i < n; ++i)      //枚举每一个物品
        for(int j = 0; j <= m; ++j) //枚举每一种可能出现的背包剩余空间
            if(j >= w[j]) dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i]);
            else dp[i][j] = dp[i-1][j];
    ans = dp[n][m];

**状态压缩：**观察状态转移方程，枚举第iii个物体时考虑的背包情况仅与放置完第i−1i-1i−1个物体时的情况有关，因此转移方程也可以转化为一维的。

dp[j]=max(dp[j],dp[j−w[i]]+v[i]) dp[j] = max(dp[j], dp[j-w[i]]+v[i]) dp[j]=max(dp[j],dp[j−w[i]]+v[i])

因为在计算dp[j−w[i]]+v[i]dp[j-w[i]]+v[i]dp[j−w[i]]+v[i]时需要前一个物品的dp[j−w[i]]dp[j-w[i]]dp[j−w[i]]，所以枚举体积时需要倒序枚举，防止覆盖上一层数据。
模板：

    for(int i = 0; i < n; ++i)      //枚举每一个物品
        for(int j = m; j >= w[i] ; --j) //枚举每一种可能出现的背包剩余空间
            dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
    ans = dp[n][m];

完全背包

与01背包相似，但是，每一个物品可以被拿无限次。
需要多枚举一个倍数的变量。
转换方程为：

dp[i][j]=max(dp[i−1][j],dp[i−1][j−k∗v[i]]+k∗w[i]) dp[i][j]=max(dp[i-1][j], dp[i-1][j-k*v[i]]+k*w[i]) dp[i][j]=max(dp[i−1][j],dp[i−1][j−k∗v[i]]+k∗w[i])

模板：

    for(int i = 0; i < n; ++i)
        for(int j = 0; j < m; ++j)
            for(int k = 1; k < j/v[i]; ++k)
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-k*v[i]]+k*w[i]);
    ans = dp[n][w];

时间复杂度O(n3)O(n^3)O(n3) 太慢了
状态压缩：

原转移方程：dp[i][j]=max(dp[i−1][j],dp[i−1][j−k∗v[i]]+k∗w[i])其中，k∈[1,jv[i]]将k展开有dp[i][j]=max(dp[i−1][j],dp[i−1][j−v[i]]+w[i],dp[i−1][j−2∗v[i]]+2∗w[i]…)同理dp[i][j−v[i]]=max(dp[i−1][j−v[i]],dp[i−1][j−2∗v[i]]+w[i],dp[i−1][j−3∗v[i]]+2∗w[i]…)两式结合有dp[i][j]=max(dp[i−1][j],dp[i][j−v[i]]+w[i])同01背包压缩掉i这个维度后为dp[j]=max(dp[j],dp[j−v[i]]+w[i]) 原转移方程：dp[i][j]=max(dp[i-1][j], dp[i-1][j-k*v[i]]+k*w[i]) 其中，k \in [1,\frac{j}{v[i]}] 将k展开有 dp[i][j]=max(dp[i-1][j], dp[i-1][j-v[i]]+w[i], dp[i-1][j-2*v[i]]+2*w[i] \ldots) 同理 dp[i][j-v[i]]=max(dp[i-1][j-v[i]], dp[i-1][j-2*v[i]]+w[i], dp[i-1][j-3*v[i]]+2*w[i] \ldots) 两式结合有 dp[i][j]=max(dp[i-1][j], dp[i][j-v[i]]+w[i]) 同01背包 压缩掉i这个维度后为 dp[j] = max(dp[j], dp[j-v[i]]+w[i]) 原转移方程：dp[i][j]=max(dp[i−1][j],dp[i−1][j−k∗v[i]]+k∗w[i])其中，k∈[1,v[i]j​]将k展开有dp[i][j]=max(dp[i−1][j],dp[i−1][j−v[i]]+w[i],dp[i−1][j−2∗v[i]]+2∗w[i]…)同理dp[i][j−v[i]]=max(dp[i−1][j−v[i]],dp[i−1][j−2∗v[i]]+w[i],dp[i−1][j−3∗v[i]]+2∗w[i]…)两式结合有dp[i][j]=max(dp[i−1][j],dp[i][j−v[i]]+w[i])同01背包压缩掉i这个维度后为dp[j]=max(dp[j],dp[j−v[i]]+w[i])

模板：

    for(int i = 0; i < n; ++i)
        for(int j = v[i]; j <= m; ++j)
            dp[j] = max(dp[j], dp[j-v[i]] + w[i]);
    ans = dp[m];
