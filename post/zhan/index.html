栈

2023-09-08

  • 定义

    栈是一种限定仅在表尾进行插入和删除操作的线性表，通常，把允许插入和删除操作的一端成为栈首，另一端称为栈尾。
    既然栈仅可以在表尾进行插入和删除，那么栈又称为先进后出的线性表。

  • 基本操作

    插入元素也叫进栈、压栈和入栈。
    删除元素也叫出栈，也叫弹栈。
    查询栈顶元素

  • 操作实例

    假设要对1，2，3这三个数字依次进栈，然后查询栈顶元素，进行一次弹栈然后再查询栈顶元素
    可以得到如下过程
  1. 依次入栈后栈内元素序列为：3，2，1.
  2. 此时查询栈顶元素，可得栈顶为3.
  3. 弹栈，此时栈内元素序列为2，1.
  4. 查询栈顶元素，可得栈顶为2.

  • 代码实现

C++的 stack头文件包含了栈的各种操作以及定义
我们可以利用其中的函数来完成之前的例子
代码及细节如下

#include<iostream>
#include<cstdio>
#include<stack>
using namespace std;

int main(){

    stack<int> s;                     //创建一个栈s，其内部元素类型为int
    
    for(int i = 1; i <= 3; i++)       //将数字1，2，3压入栈内
        s.push(i);                    //s.push(i)是将i压入栈顶
                                      //s.top()可返回栈顶元素
    printf("当前栈顶元素为：%d\n",s.top());

    s.pop();                          //进行一次弹栈

    printf("进行一次弹栈后栈顶元素：%d\n",s.top());

    while(!s.empty())                 //s.empty()用于检测栈是否为空栈 若为空栈返回true
    {
        printf("%d ",s.top());
        s.pop();
    }                                 //查看栈内元素及顺序
    
    return 0;
}

  • 数组写法

#include<iostream>
#include<cstdio>
using namespace std;

int pos = 0;
int s[105];

int top(){

    return s[pos];
}

bool empty(){

    if(pos)
        return false;
    else
        return true;
}

void pop(){

    s[pos--] = 0;
}

void push(int x){

    s[++pos] = x;

}

int main(){

    for(int i = 1; i <= 3; i++)
        push(i);
    
    printf("当前栈顶元素为：%d\n",top());

    pop();

    printf("进行一次弹栈后栈顶元素：%d\n",top());

    while(!empty())
    {
        printf("%d ",top());
        pop();
    }
    
    return 0;
}
