  • If c0≥c1c_0 \geq c_1c0​≥c1​, the majority is 0.
  • If c0<c1c_0 < c_1c0​<c1​, the majority is 0.

We need to determin whether we could make the sequence to [1][1][1].

Soultion

We could greedily do the operations, in for each subsequences with all 0, we could greedily transform the into [0][0][0]. Then we can count the number of ones and zeros in the sequence, if the number of ones is greater than zeros then the sequece could be transform to [1][1][1], otherwise we can't achieve that.

Non-rigorous proof of correctness

In each operation if we want to decrease the number of zeros will const more numbers of ones. Therefore if after we transform all the subsequences which only consists 000 to [0][0][0] is definitely beneficial to transform the entire sequence to [1][1][1], the number of ones is still smaller than the number of zeros, then we will never transform the sequence to [1][1][1].
